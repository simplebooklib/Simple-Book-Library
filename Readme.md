Simple Book Library
===================

Spring Boot 기반으로 개발되어 있습니다. BookLibraryApplication 클래스를 실행해서 확인할 수 있습니다.
메모리 DB 기반으로 실행되기 때문에 재시작시, 북마크, 최근검색 기록들은 사라집니다.

최초 실행시 data.sql로 기본 사용자 정보가 등록되고, 로그인 페이지에서 미리 계정이 입력되어 있습니다.

주요 dependencies
 - Spring Boot
 - Spring Security
 - Spring Data JPA
 - Sitemesh
 - EhCache
 - Lombok
